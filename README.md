# 项目简介

## 核心功能
- 基于 YOLOv8 的路面缺陷实例分割检测，支持裂缝、坑洼和路面三类缺陷识别
- 提供完整源码、1500 张标注数据集、训练好的模型及训练结果
- 支持多种输入方式：单张图片、图片文件夹、视频文件和本地摄像头
- 实时显示检测结果，包括目标位置、目标总数和置信度
- 支持自定义置信度和 IOU 阈值
- 提供界面选择显示标签和原图
- 支持选择检测模型
- 支持批量检测并查看所有结果
- 支持检测结果保存

## 适用场景
- 道路维护与检测
- 市政设施管理
- 交通基础设施评估
- 自动化道路质量分析

## 技术特点
- 使用 Pycharm/VSCode + Anaconda 开发环境
- 基于 Python 3.9，集成 OpenCV、PyQt5、Ultralytics 和 Torch1.9 等技术栈
- 提供完整 UI 界面源文件和图标
- 配套项目运行教程，确保环境配置后可直接使用
